INSTALLATION GUIDE

1. EXTRACT Psyplay v1.1.7.zip
2. GO TO YOURSITE/WP-ADMIN/THEMES.PHP
3. ADD THEMS AND UPLOAD Grifus.zip AND ACTIVATE IT
4. GO TO MUNDOTHEMES ADD LICENSE KEY Mundothemes CLICK SAVE CHANGES 
THEN ACTIVATE LICENSE KEY
5. GO TO YOURSITE/WP-ADMIN/THEMES.PHP
6. ADD THEMES AND UPLOAD PSYPLAY.ZIP AND ACTIVATE IT
7. EXTRACT ACF.RAR
8. GO TO YOURSITE/WP-ADMON/CUSTOM-FIELDS/TOOLS/
9. IN IMPORT FIELD GROUPS CLICK CHOOSE FILE BUTTON THEN LOCATE THE acf-export-2017-10-12.json
10. THEN CLICK IMPORT
11. WELL DONE :)

PS: TO ACTIVATE THE AUTOMATIC EMBED ON MOVIES AND EPISODES PLEASE ACTIVATE THE FAKE PLAYER
DON'T ADD PLAYER IN EVERY MOVIES AND EPISODES
